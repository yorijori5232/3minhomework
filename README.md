# 3분 집안일

미뤄둔 집안일을 가볍게 시작하게 만드는 웹 서비스. 즉석식품 '3분 카레'를 모티프로, 봉지를 뜯으면 오늘 할 집안일 하나가 나온다.

## 사용자 흐름

접속 → 난이도 선택 → 봉지 뜯기 → 미션 + TIP 확인 → 시작하기(타이머) → 완료했어요 → 공유 / 하나 더 뽑기

## 구성

| 파일 | 설명 |
| --- | --- |
| `index.html` | 화면·인터랙션·타이머 전체 (의존성 없는 단일 파일) |
| `missions.js` | 미션 200개 데이터 (순한맛 139 / 매운맛 61) |
| `assets/` | 봉지 패키지 이미지 (순한맛 / 매운맛) |

## 난이도

원본 미션 데이터의 난이도는 순한맛 / 중간맛 / 매운맛 3단계지만, 서비스는 소요시간 3분을 기준선으로 2단계로 운영한다.

- **순한맛** — 원본 순한맛 + 중간맛, 1~3분, 139개
- **매운맛** — 원본 매운맛, 5~12분, 61개

## 로컬에서 보기

경로가 모두 상대 경로라 `index.html`을 브라우저로 바로 열어도 되지만, 로컬 서버로 띄우는 쪽을 권장한다.

```
python3 -m http.server 4173
```

## 미션 추가하기

`missions.js`의 `mild` / `hot` 배열에 항목을 추가한다.

```js
{ s: '주방', t: '3분', c: '설거지', m: '싱크대에 쌓인 컵 전부 씻기', tip: '컵만 먼저 씻어도 싱크대가 절반은 비어요.' }
```

`s` 공간 태그 · `t` 예상 소요(타이머가 이 값을 읽는다) · `c` 카테고리 · `m` 미션 · `tip` 실행 팁
